# 学生アルバイト給与計算システム　計算ロジック仕様書

## 1. 本書の目的

本書は、学生アルバイト給与計算における全業務区分の計算ロジックを明文化し、システム実装の基礎仕様とするものである。現状整理書（As-Is）で整理した給与体系に基づき、入力値・計算式・出力値を網羅的に定義する。

## 2. 端数処理ルール

| **項目** | **ルール** | **備考** |
|---|---|---|
| 端数処理方式 | 切り上げ（ROUNDUP） | 1円未満を切り上げ |
| 適用タイミング | 月次合計時に1回 | 各項目の計算途中では丸めず、月の総支給額算出時に最終切り上げ |

> ※ 各項目の計算途中では端数を保持したまま加算し、月次の総支給額を算出した時点で1円未満を切り上げる。

## 3. 業務区分別 計算ロジック

### 3.1 業務A: オフライン自習室（時給制）

データソースが「入退くん」と「日報」の二元構成である点に注意。

| **項目** | **計算式** | **備考** |
|---|---|---|
| 自習室勤務 | 時給1,500円 ÷ 60 × 勤務分数 | 1分単位。データソース: 入退くん |
| ヘルプ勤務 | 時給1,500円 ÷ 60 × 勤務分数 | 他校ヘルプ時。データソース: 日報 |
| 研修・事務作業 | 時給1,226円 ÷ 60 × 勤務分数 | データソース: 日報 |

**計算例:** 14:00〜17:23勤務（自習室業務）の場合
- 勤務分数 = 3時間23分 = 203分
- 報酬 = 1,500 ÷ 60 × 203 = 5,075.00円

### 3.2 業務B: オンライン塾 Aサービス（コマ給・1コマ50分）

手当の種類が最も多い業務区分。1コマ = 50分。

| **項目** | **計算式** | **備考** |
|---|---|---|
| 面談（通常） | 1,250円 × 実施コマ数 | 1コマ = 50分 |
| 面談（当日キャンセル） | 1,021.65円 × キャンセルコマ数 | |
| 面談手当 | 103円 × 実施コマ数 | キャンセルコマは対象外 |
| 勤務手当 | 103円 × 面談実施日数 | 面談があった日のみ |
| 保護者メッセージ記入 | 205円 × 対象コマ数 | |
| 研修・事務作業 | 時給1,226円 ÷ 60 × 勤務分数 | |
| 深夜割増 | (1,250円÷50分) × 22時以降分数 × 0.25 | コマ給を時給換算（25円/分）して割増 |

**計算例:** 実施3コマ＋当日キャンセル1コマ、保護者MSG 2コマ分、2日間勤務の場合
- 基本報酬 = (1,250 × 3) + (1,021.65 × 1) = 4,771.65円
- 面談手当 = 103 × 3 = 309円（キャンセル1コマは対象外）
- 勤務手当 = 103 × 2 = 206円（2日間とも面談実施あり）
- 保護者MSG = 205 × 2 = 410円
- 小計 = 5,696.65円（端数は月合計時に処理）

## 4. 共通手当・割増の計算ロジック

| **項目** | **計算式** | **備考** |
|---|---|---|
| 交通費（通常） | 登録額 × 出勤日数 | オンライン作業・研修のみの日は除外 |
| 交通費（例外） | 日報申請額をそのまま加算 | ヘルプ勤務・単発で金額変動時 |
| 深夜割増（コマ給業務） | (コマ給÷コマ所定分数) × 22時以降分数 × 0.25 | 一律コマ給ベース（労働者有利）。10分超でアラート |
| 休業手当 | 1,226円 ÷ 60 × カットされたシフト分数 | 会社都合キャンセル。ソース: SS |
| 休日割増 | ベース時給 × 休日勤務分数 ÷ 60 × 0.35 | 法定通り（35%）。現状ほぼ発生しない |
| プロインセンティブ | 1,000円（対象者のみ） | 対象者リストにより毎月変動 |

### 4.1 深夜割増の詳細

**対象:** 22:00以降の勤務分数に対して25%の割増賃金を加算する。

**22時以降の分数算出:**
```
22時以降分数 = 日報の終了時刻 − 22:00
```
※ 日報には全体の開始・終了時刻のみ記録されており、個々のコマの時刻情報はない。

**ベース時給の決定ルール:**

日報上で面談と研修の時間帯が分離できないため、コマ給業務（B〜F）では一律コマ給の時給換算額をベースとする（労働者有利）。研修のみが22時以降にかかっている場合でもコマ給ベースで計算する。

**【アラート条件】** 22時以降の分数が10分を超える場合、担当者に確認アラートを出力する。大幅な深夜勤務については人間が内容を確認し、必要に応じて手動補正する運用とする。

**業務別の時給換算額:**

| 業務 | 計算式 | 時給換算額 |
|---|---|---|
| 業務B | 1,250円 ÷ 50分 | 25.00円/分 |
| 業務C | 800円 ÷ 30分 | 26.67円/分 |
| 業務D | 500円 ÷ 20分 | 25.00円/分 |
| 業務E | 625円 ÷ 25分 | 25.00円/分 |
| 業務F | 750円 ÷ 25分 | 30.00円/分 |

**計算式（コマ給業務共通）:**
```
深夜割増額 = 時給換算額（円/分） × 22時以降分数 × 0.25
```

**計算例:** 業務B、日報の終了時刻が22:20の場合
- 22時以降分数 = 22:20 − 22:00 = 20分
- 深夜割増 = 25.00 × 20 × 0.25 = 125円
- → 10分超のため確認アラート出力

> ※ 業務A（オフライン自習室）は深夜割増の対象外（22時以降の稼働なし）。研修・事務作業が22時以降にかかる場合も、コマ給業務と同日であればコマ給ベースで計算する。

## 6. 月次総支給額の算出

個人ごとの月次総支給額は、以下の項目を合算し、最終的に1円未満を切り上げて算出する。

| **#** | **項目** | **内容** |
|---|---|---|
| ① | 基本報酬合計 | 業務ごとの報酬（通常コマ＋キャンセルコマ＋時給分）の合計 |
| ② | 業務固有手当合計 | 面談手当＋コマ手当＋勤務手当＋保護者MSG＋tel発信の合計 |
| ③ | 研修・事務作業合計 | 時給1,226円 × 合計勤務分数 ÷ 60 |
| ④ | 深夜割増合計 | 時給制分＋コマ給分の深夜割増の合計 |
| ⑤ | 交通費合計 | (登録額×出勤日数) ＋ 日報申請の例外交通費 |
| ⑥ | 共通手当合計 | 休業手当＋プロインセンティブ |
| ⑦ | 調整金額 | コンプラテスト＋新人研修＋有給＋紹介料 |
| | **月次総支給額** | **ROUNDUP( ① + ② + ③ + ④ + ⑤ + ⑥ + ⑦ )** |

> ※ 1人のアルバイトが複数業務を兼務する場合、①〜④は業務ごとに算出してから合算する。
>
> ※ ROUNDUPは月次総支給額に対して1回のみ適用する。各項目の計算途中では丸めない。

## 8. 例外処理・エッジケース

1. **業務Dのコマ手当:** 唯一、当日キャンセルコマにもコマ手当（103円）が適用される。他業務（B, C, E, F）ではキャンセルコマは手当対象外。実装時に業務Dのみ分岐が必要。
2. **業務Aのデータソース分岐:** 通常勤務は入退くん、ヘルプ・研修は日報。同一人物の同月データを2つのソースから統合する処理が必要。
3. **交通費の重複:** ヘルプ勤務と通常勤務が同一日に重なることは原則ない。万が一発生した場合は単発交通費（日報申請額）を優先し、マスタ登録額は適用しない。
4. **1人が複数業務を兼務:** 同一月に複数の業務区分で勤務した場合、業務ごとに報酬・手当を個別計算し、月次合計時に一括で端数切り上げする。
5. **アルバイト紹介料の金額変動:** キャンペーンとして随時変動するため固定ルールはない。その都度の適用単価をスプレッドシート上で指定し、調整金額として計上する。
6. **深夜割増のベース時給判定:** コマ給業務では面談・研修の時間帯が日報上で分離できないため、一律コマ給の時給換算額で計算する（労働者有利方式）。22時以降の分数が10分を超える場合は確認アラートを出力し、担当者が内容を確認して必要に応じ手動補正する。
7. **深夜割増の分数算出:** 日報の終了時刻から22:00を引いて算出する。コマごとの時刻情報は日報にないため、日報全体の終了時刻ベースで一括計算する。
